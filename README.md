# chatroom


A chatroom in GO.


Thanks to blog: https://medium.com/@nqbao/writing-a-chat-server-in-go-3b61ccc2a8ed


cli available



#start a chat server

  ./chatroom server :3333


#start a client

  ./chatroom client :3333


#commands available


#send message

  m hello world!

  msg verbose world!


#set username

  n user1

  name verbose_user1


#exit

  e

  exit
